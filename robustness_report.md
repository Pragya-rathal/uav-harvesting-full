# Statistical Robustness Report

## Seed sweep support

The publication suite runs seed sweeps over:

- seeds 0–9 by default

## Statistics computed

For each metric, the suite computes:

- mean
- standard deviation
- minimum
- maximum
- 95% confidence interval

## Output files

- `performance_summary.csv`
- `robustness_runs.csv`
- `robustness_summary.json`
- `robustness_report.md` (this file can be extended with final numeric results)
